# Raw Insurance Datasets

This folder contains the original raw datasets used as source systems
for the insurance data warehouse.

These files represent extracted data from operational systems
and should not be modified.

## Datasets
- dim_customer.csv
- dim_policy.csv
- dim_product.csv
- dim_agent.csv
- dim_location.csv
- dim_claim_type.csv
- dim_date.csv
- fact_claims.csv
