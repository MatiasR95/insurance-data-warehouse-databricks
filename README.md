# Insurance Data Warehouse on Databricks

## Overview
This project demonstrates the design and implementation of an end-to-end insurance data warehouse using Databricks, Delta Lake, and Spark SQL.

## Business Context
The warehouse supports insurance claims analytics, including:
- Claim frequency and severity
- Paid vs incurred analysis
- Trends by product, geography, and time

## Architecture
- Medallion Architecture (Bronze, Silver, Gold)
- Star Schema in Gold Layer
- Delta Lake tables

## Dataset
Synthetic insurance data including:
- Claims
- Policies
- Customers
- Products
- Agents

## Tools & Technologies
- Databricks (Spark SQL)
- Delta Lake
- Git & GitHub
